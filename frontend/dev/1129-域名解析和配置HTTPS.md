# 域名解析和配置 HTTPS

前提你已经购买域名并且通过备案

## 步骤

> 阿里云公告：.com/.net/.cn/.xin/.top/.xyz/.vip/.club/.shop/.wang/.ren 等域名注册成功后必须进行域名实名认证，否则会造成解析不生效，实名认证审核通过后的 1-2 个工作日解析可恢复使用
