### 字体形状

（原理 根据像素得到坐标）

| 序号 | 步骤                                            |
| :--: | :---------------------------------------------- |
|  1   | UI 给出图片                                     |
|  2   | 前端读取图片每一个像素                          |
|  3   | 去掉部分坐标                                    |
|  4   | 把坐标作为一个点画出 点有大小 比如 5 像素小圆圈 |
|  5   | 把坐标渲染出来就得到 字体形状                   |
|  6   | 把坐标打乱随机在一定范围渲染，就得到乱序的过程  |
